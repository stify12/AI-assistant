# 基准效果存储目录

此目录用于存储AI批改评估的基准效果数据。

## 文件命名规则

文件名格式：`{作业名称}_{页码}.json`

例如：
- `英语作业第一单元_1.json`
- `数学练习册_23.json`

## 文件结构

```json
{
  "homework_name": "作业名称",
  "page_num": "页码",
  "subject_id": 0,
  "base_effect": [
    {
      "index": "1",
      "answer": "A",
      "userAnswer": "A",
      "correct": "yes",
      "tempIndex": 0
    }
  ],
  "created_at": "2026-01-08T15:52:00.000000",
  "homework_id": "作业ID"
}
```

## 功能说明

1. **自动识别**：点击"自动识别"按钮，系统会从数据库获取作业图片，调用1.8模型识别并自动保存基准效果
2. **自动加载**：选择作业时，如果存在相同作业名+页码的基准效果文件，会自动加载
3. **手动保存**：编辑基准效果后，点击"保存基准效果"按钮手动保存
4. **复用机制**：相同作业+页码的基准效果可以在不同批改记录中复用

## 注意事项

- 文件名中的非法字符（如 `<>:"/\|?*`）会被替换为下划线 `_`
- 基准效果数据以JSON格式存储，便于查看和编辑
- 建议定期备份此目录
