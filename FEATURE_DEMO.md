# 批量评估学科功能演示

## 功能亮点

### 🎯 智能任务命名
选择学科后，任务名称自动生成为 **学科-月/日** 格式，简洁明了。

**示例：**
- 选择"数学" → 自动生成 `数学-1/15`
- 选择"物理" → 自动生成 `物理-1/15`
- 不选学科 → 自动生成 `批量评估-1/15`

### 🔍 快速学科筛选
任务列表顶部一键筛选，快速找到目标学科的评估任务。

### 🏷️ 清晰学科标签
任务列表中每个任务显示学科标签，一目了然。

## 使用场景

### 场景1：日常评估
```
1. 点击"新建任务"
2. 选择学科"数学"
3. 任务名称留空（自动生成：数学-1/15）
4. 选择今天的数学作业
5. 点击"创建任务"
```

### 场景2：查找历史任务
```
1. 在任务列表顶部选择"数学"
2. 列表自动筛选，只显示数学任务
3. 快速找到需要的评估记录
```

### 场景3：多学科管理
```
任务列表显示：
- [数学] 数学-1/15 (已完成 85.2%)
- [物理] 物理-1/14 (已完成 78.9%)
- [英语] 英语-1/13 (评估中...)
```

## 设计理念

### 自动化优先
- 减少手动输入，提升效率
- 统一命名规范，便于管理

### 灵活可控
- 支持手动输入覆盖自动命名
- 学科选择可选，不强制

### 向后兼容
- 旧任务数据无缝兼容
- 不影响现有功能

## 技术特点

- **零学习成本**：符合直觉的交互设计
- **高性能**：前端筛选，响应迅速
- **可扩展**：学科映射易于扩展
- **数据安全**：只增加字段，不修改原有数据
